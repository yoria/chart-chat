# chart-chat
株に関する意見交換のためのコミュニケーションサイトです。一応趣旨としては、株やFXの乱高下に対するリアルタイムな反応をして欲しいため、コメントのリアルタイム性を意識しました。

# Type
Web App (Webアプリケーション)

# 対応デバイス
- PC  
- スマートフォン  

# Go to site (Webサイトのリンク)
1. https://chart-chat.web.app/
2. https://chart-chat.firebaseapp.com/  
今でも動きます！

# Technical Specification (技術仕様)
- HTML CSS  
- javaScript (Vue.js)  
- Node.js  
- Firebase


# Dependencies (依存関係)
- Google Firebase
- TradingView API
